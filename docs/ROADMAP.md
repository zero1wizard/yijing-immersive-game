# ROADMAP

## v1 MVP

- 完成 64 卦内容底库与六大章节路图
- 打通遭遇、选择、转卦、解锁、札记、日课
- 建立本地持久化与核心逻辑测试
- 输出产品、内容、路线三份基础文档

## v2

- 为每一卦增加更丰富的分支事件与多轮选择
- 加入原典摘句、译注和术语对照阅读层
- 加入音景、章节过场与更强的氛围化视觉叙事
- 做成“修习档案”：周回顾、主题统计、学习建议
- 支持分享某次推演记录或导出个人札记
- 把“今日一卦”扩展为连续 7 天的反思路径
- 增加标签化搜索，如“创业”“关系”“修身”“决策”

## v3

- 引入可配置的课程路径，例如“创业局”“关系局”“领导局”
- 做多人讨论房间，让同一卦可被不同人共同注释
- 提供更精细的内容编辑器，方便后续策展和扩写
- 增加跨设备同步、账号体系和个人学习时间线

## 当前已知缺口

- 目前每卦仍采用统一生成模板，深度事件数量有限
- 尚未接入原典、译注和出处管理能力
- 没有云端同步、导出和协作能力
- 尚未建立内容运营后台与审校流程
